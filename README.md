# AutoClicker

A simple Python CLI to automatically click mouse buttons or press keys.

## Usage

Run `main.py` with the option `-h`/`--help` to see how it works.

## Default values

|   **Option**    |  **Value**  |
|:---------------:|:-----------:|
| `-m`/`--mouse`  | Button.left |
|  `-k`/`--key`   |   *None*    |
| `-t`/`--toggle` |      X      |
|  `-x`/`--exit`  |      Z      |
| `-d`/`--delay`  |      1      |
